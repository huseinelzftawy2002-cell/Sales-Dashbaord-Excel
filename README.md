📊 Sales Analysis Dashboard (Excel)
📌 Overview

This project showcases an interactive Sales Analysis Dashboard built using Microsoft Excel.
The dashboard transforms raw sales data into meaningful insights, enabling better understanding of business performance across multiple dimensions.

It is designed to support data-driven decision-making by providing a clear and dynamic view of key metrics.

🎯 Objectives
Analyze overall sales performance
Track profit and margin trends
Identify top and bottom performing products
Understand customer distribution across states
Enable interactive exploration using filters
📊 Key Features
💰 KPIs Overview
Total Sales
Total Profit
Total Quantity
Average Discount
Profit Margin
📈 Category Analysis
Sales, Profit, and Quantity by Category
Profit breakdown by Subcategory
🏆 Product Performance
Top 10 Products by Sales
Bottom 10 Products by Sales
🌍 Geographical Insights
Total Customers by State (Map Visualization)
🎛️ Interactivity
Dynamic slicers for:
Year
Quarter
Category
🛠️ Tools & Techniques
Microsoft Excel
Pivot Tables
Pivot Charts

Data Cleaning & Transformation
KPI Design
📂 Dataset Description

The dataset includes the following fields:

Order Date
Sales
Profit
Quantity
Discount

💡 Insights Gained
Technology category generates the highest sales
Certain subcategories contribute more to profit than others
A small number of products drive the majority of revenue
Customer distribution varies significantly across states
